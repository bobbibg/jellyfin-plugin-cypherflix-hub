Setup

Cypherflix Hub renders Discover and Manage views into anchor <div>s hosted by the Custom Tabs plugin. Sidenav links are added via the KefinTweaks plugin's Custom Menu Links feature. Configure those plugins as below; this plugin will detect the anchor divs and render content into them.
  1. Open Dashboard → Plugins → Custom Tabs and add two tabs:
    • Title Discover, Body <div class="sections cypherflix-discover"></div>
    • Title Manage, Body <div class="sections cypherflix-manage"></div>
  2. Open KefinTweaks → Custom Menu Links and add drawer entries pointing at #/home?tab=N for the indexes Custom Tabs assigned (Home is 0, Favourites is 1, your custom tabs are 2 and onward).
  3. Save. Refresh the home page. Discover and Manage will render inside their respective tabs.
  4. For book metadata + covers: set HARDCOVER_API_TOKEN in the cypherflix-grabber container's environment. Free tokens at hardcover.app/account/api. Without it, Trending Books returns empty and book covers won't auto-populate.

Settings

Base URL of cypherflix-grabber V2 (e.g. http://192.168.1.165:7960). No trailing slash.
Shared secret matching CYPHERFLIX_API_TOKEN on the backend. Leave blank if no token is configured.

Creator blocklist

Anything any of these creators had involvement with — books, comics, manga, movies, TV — is hidden from Discover, Search, and Coming Soon. Items already in your library / queue stay in place. Aliases catch pseudonyms (e.g. add Robert Galbraith alongside J.K. Rowling).
