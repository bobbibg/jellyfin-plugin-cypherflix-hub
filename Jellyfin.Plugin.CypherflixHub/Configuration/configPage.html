Cypherflix Hub

Configure providers (Jellyseerr, Readarr, ReadMeABook, …) and the Meilisearch instance used for unified search. Each provider type can have multiple instances — e.g. one Readarr for books and one for comics.

Until the provider UI lands (Task UI-002), this page is a placeholder. Edit settings via plugins/CypherflixHub.xml in the Jellyfin config dir or wait for the next release.
